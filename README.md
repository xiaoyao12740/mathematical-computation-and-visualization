# 数学计算与可视化练习集

这是一个基于 MATLAB 的数学计算与可视化练习仓库，收录了若干相对独立的小脚本，内容涵盖：

- 常微分方程与边值问题数值求解
- 符号微分方程求解
- 傅里叶级数近似与函数可视化
- 回归分析与数据拟合
- 数学基础训练与图形绘制

本仓库更接近“数学计算与可视化方法练习集”，适合作为 MATLAB 学习记录、课程练习归档以及 GitHub 作品集中的基础型项目展示。

---

## 项目简介

在数学建模、数值分析和科学计算学习过程中，很多问题并不一定构成一个庞大的完整工程，但它们能够较好地体现以下能力：

- 将数学问题转化为程序表达
- 使用 MATLAB 进行数值求解
- 对结果进行图形化展示
- 结合符号计算与数值计算进行分析

本仓库即围绕这些目标，整理了若干 MATLAB 小脚本。虽然每个脚本规模不大，但整体覆盖了多个常见方向，包括微分方程、函数逼近、回归建模、几何绘图与基础算法练习。

## 各文件说明

---

### 一、微分方程与边值问题模块

这一部分主要展示 MATLAB 在常微分方程、边值问题以及符号求解方面的基础应用。

#### 1. `rigid.m`

该脚本使用 MATLAB 的 `bvp4c` 求解一个边值问题，并绘制数值解曲线。

其整体思路是：

* 将二阶微分方程改写为一阶方程组
* 给定边界条件
* 使用 `bvpinit` 提供初始猜测
* 使用 `bvp4c` 进行数值求解
* 最终调用 `deval` 对解进行插值并绘图

这是一个典型的**边值问题数值求解练习**。

---

#### 2. `odefun.m`

这是与 `rigid.m` 配套使用的微分方程定义函数。

在该文件中，方程被写成一阶系统形式：

* `y(1)` 表示原函数
* `y(2)` 表示一阶导数

该函数返回状态变量的导数，用于传递给 `bvp4c`。

---

#### 3. `bcfun.m`

这是与 `rigid.m` 配套使用的边界条件函数。

当前设定的边界条件为：

* `y(0)=0`
* `y'(0)=0`

它与 `odefun.m` 一起构成完整的边值问题求解框架。

---

#### 4. `daodan.m`

该脚本使用 MATLAB 符号工具箱定义符号函数 `y(x)`，求导并尝试通过 `dsolve` 对微分方程进行解析求解。

这个脚本更偏向于：

* 符号建模
* 微分方程解析形式尝试
* 初值条件设定
* 解析解展示

它与 `rigid.m` 形成了一个有意思的对照：

* `daodan.m` 更偏解析求解
* `rigid.m` 更偏数值求解

这能够体现 MATLAB 在不同求解思路下的使用方式。

---

### 二、函数逼近与可视化模块

#### `fuliyebest21.m`

该脚本以方波函数为例，计算不同截断阶数下的傅里叶级数部分和，并绘制原函数与近似曲线。

脚本特点包括：

* 定义分段函数（方波）
* 选择多个不同的截断项数
* 逐步构造傅里叶级数部分和
* 将原函数与不同阶数近似曲线同时绘制

该脚本适合展示：

* 傅里叶级数逼近思想
* 随截断阶数增加，近似效果如何变化
* 数学分析中的“函数展开”如何通过程序直观呈现

这是一个典型的**数学分析 + 可视化展示**练习。

---

### 三、数据拟合与回归分析模块

#### `yagao.m`

这是当前仓库中内容相对丰富的一个脚本，主要围绕一组实验数据进行拟合、回归分析与可视化。

从代码结构看，它包含以下几部分内容：

1. **一元线性拟合**

   * 对 `x1` 与 `y` 的关系进行一次多项式拟合
   * 绘制散点图和拟合曲线

2. **一元二次拟合**

   * 对 `x2` 与 `y` 的关系进行二次拟合
   * 绘制数据点与拟合曲线

3. **多元回归分析**

   * 使用 `regress` 建立包含常数项、一次项、二次项以及交互项的回归模型
   * 对不同模型形式进行比较

4. **回归结果可视化**

   * 根据给定系数构造预测表达式
   * 绘制不同情形下的预测曲线
   * 使用 `rstool` 做进一步回归可视化分析

从整体上看，这个脚本展示了 MATLAB 在以下方面的基础能力：

* 多项式拟合
* 回归分析
* 交互项建模
* 散点图与拟合曲线绘制
* 回归工具辅助分析

它比较适合作为一个**数据分析与拟合建模练习案例**。

### 四、数学训练与基础绘图模块

该部分位于 `trainhomework/` 文件夹，更偏向训练性质的小作业或基础练习。

---

#### 1. `trainhomework/tailezhanaiqiupi.m`

该脚本通过莱布尼茨级数近似计算圆周率：

[
\pi \approx 4\sum_{n=1}^{N}\frac{(-1)^{n+1}}{2n-1}
]

用户输入迭代精度（项数）后，程序输出对应的近似值。

该脚本适合展示：

* 无穷级数数值逼近
* 数学公式到程序循环实现的转化
* 基础数值计算练习

---

#### 2. `trainhomework/thetaround.m`

该脚本用于绘制圆。

代码中采用了两种思路：

* 参数方程法：
  [
  x=r\cos\theta,\quad y=r\sin\theta
  ]
* 上下半圆函数法：
  [
  y=\pm \sqrt{1-x^2}
  ]

该脚本体现了同一几何对象可以通过不同数学表达方式来绘制，也适合作为 MATLAB 基础绘图练习。

---

#### 3. `trainhomework/xiugaijvzhen.m`

该脚本实现了一个矩阵扩展小功能：

* 用户输入原矩阵
* 程序在矩阵外层补一圈 0
* 输出新的扩展矩阵

该脚本虽然简单，但能够体现：

* 矩阵维度处理
* MATLAB 索引操作
* 基础数组构造

属于很典型的**矩阵操作入门练习**。
